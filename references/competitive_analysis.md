# 竞品对比分析

## 技术路线对比总览

| 维度 | RFID无源无线（悦和） | 荧光光纤测温 | SAW声表面波 | 有源无线 |
|------|---------------------|-------------|------------|---------|
| 供电方式 | 无源（环境取能） | 光纤供电 | 无源（射频） | 电池 |
| 测温范围 | -40~150℃ | -40~300℃ | -40~150℃ | -40~125℃ |
| 测温精度 | ±1℃ | ±0.5℃ | ±1℃ | ±0.5℃ |
| 安装方式 | 非接触/粘贴/扎带 | 接触式布线 | 接触式 | 粘贴/扎带 |
| 测点密度 | 高（单采集器96+点） | 低（4~16通道） | 中 | 中 |
| 改造友好度 | ⭐⭐⭐⭐⭐ | ⭐⭐ | ⭐⭐⭐ | ⭐⭐⭐⭐ |
| 维护需求 | 免维护 | 光纤易老化需维护 | 免维护 | 电池需更换 |
| 耐温能力 | 225℃（传感器本体） | 300℃+ | 150℃ | 85℃（电池限制） |

## 详细对比

### vs 荧光光纤测温

| 指标 | RFID（悦和） | 荧光光纤 |
|------|-------------|---------|
| 安装效率 | **20分钟/柜** | 40分钟/柜 |
| 多点同步 | **96+点/采集器** | 4~16通道 |
| 免布线 | ✅ 完全无线 | ❌ 需要光纤布线 |
| 改造项目 | ✅ **极适合改造** | ❌ 布线困难 |
| 高温场景 | △ 最高150℃ | ✅ **可达300℃以上** |
| 数据一致性 | **高** | 低（光纤衰减） |
| 免维护 | ✅ **是** | ❌ 否（光纤老化） |

**结论**：对开关柜/母线槽等大量测点场景，RFID成本更低、安装更快。对超高温（>150℃）场景选荧光光纤。

### vs CT取电式无线测温

| 指标 | RFID（悦和） | CT取电无线 |
|------|------------|-----------|
| 安全隐患 | ✅ **安全，无源取能** | ❌ 大电流CT取电有安全风险 |
| 抗电磁干扰 | ✅ **强（CMOS单芯片）** | ❌ 受电磁场影响大 |
| 使用寿命 | ✅ **与设备同寿命** | ❌ CT易饱和、需定期更换 |
| 安装复杂度 | 低（无线） | 中（需穿CT） |
| 适用场景 | 动触头/母排/电缆头 | 仅限载流导线 |

**结论**：RFID无源方案在安全性、可靠性上全面优于CT取电方案，特别适合改造项目和高压环境。

### vs 有源无线（LoRa/NB-IoT）

| 指标 | RFID（悦和） | 有源无线 |
|------|------------|--------|
| 供电 | ✅ **无电池，环境取能** | ❌ 电池供电 |
| 维护 | ✅ **免维护** | ❌ **需定期换电池** |
| 寿命 | ✅ **10年+** | ⚠️ 电池寿命2~5年 |
| 批量读取 | ✅ **支持（25ms采样）** | ❌ 点对点，逐个读取 |
| 防护等级 | ✅ **IP65~IP68** | △ IP54~IP65 |

**结论**：有源无线适合分散的单点监测；RFID适合高密度、高可靠、低维护的工业场景。

## 选型决策树

```
是否有现成布线？
  │
  ├─ 是 → 预算充足 + 高温（>150℃）→ 荧光光纤
  │        预算有限 + 大量测点 → RFID无源
  │
  └─ 否（改造）→ RFID无源 ✅
        └─ 是否高压环境？→ 是 → RFID ✅（安全优势明显）
                          否 → RFID或有源均可
```
